Apache Thrift
RPC Framework
Python 2.7

To compile & run the code:
1. chmod +x init
2. ./init node.txt 
	## node.txt contains all nodes in form of ip:port form
3. ./server PORT#
4. python TEST_CLIENT_FILE IP:PORT
